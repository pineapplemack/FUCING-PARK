# 🎢 操逼乐园 - 3D 探索大冒险 (Funfair 3D Explorer)

欢迎来到 **操逼乐园 (V2.3.0 写实画质 PRO 版)**！这是一个基于 Three.js (WebGL) 打造的第三人称 3D 游乐园探索游戏。游戏内置了写实级环境光影、丰富的互动打工赚钱系统、8项具有暗示性 (NSFW) 几何造型的写实级游乐设施，以及 45 个在乐园中自主漫游巡路的 3D 繁华 NPC。

## 🕹️ 键盘操作说明

- `W` `A` `S` `D` 或 `↑` `↓` `←` `→`：控制角色移动 / 驾驶载具
- `Space`（空格键）：控制角色跳跃
- `鼠标移动`：旋转相机视角
- `E` 键：
  - 在门口闸机处：购买门票入园
  - 在打工店前：开始打工
  - 在设施前：花费 $20 乘坐设施
  - 在乘坐设施时：安全下车
- `V` 键：乘坐过山车时切换（第一人称 / 第三人称 / 电影级）视角

## 🚀 本地运行与发布指南

### 本地直接运行
由于 Three.js 已实现本地化下载，您无需搭建任何服务器。
1. 下载整个项目文件夹。
2. 双击运行 **`index.html`**，即可直接在任何现代浏览器（Chrome, Edge, Firefox, Safari 等）中流畅游玩。

### 发布到 GitHub Pages（在线网页）
1. 在您的 GitHub 账号下创建一个全新的 Public 仓库，例如命名为 `funfair-game`。
2. 点击仓库主页的 `uploading an existing file`。
3. 将本项目文件夹中的 4 个文件（`index.html`、`style.css`、`game.js`、`three.min.js`）一起拖拽上传并 Commit。
4. 前往该仓库的 **`Settings` -> `Pages`**，在 Branch 选项中选择 **`main`** (或 `master`)，然后点击 **`Save`**。
5. 等待 1 分钟，您的游戏就会发布在 `https://您的用户名.github.io/funfair-game/` 上，可以发送给任何人在线畅玩！
